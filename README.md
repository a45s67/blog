# Fish's Blog

Personal blog built with [Astro](https://astro.build) using the [astro-cactus](https://github.com/chrismwilliams/astro-theme-cactus) theme.

## Commands

| Command          | Action                                                         |
| :--------------- | :------------------------------------------------------------- |
| `npm install`    | Install dependencies                                           |
| `npm run dev`    | Start local dev server at `localhost:4321`                      |
| `npm run build`  | Build production site to `./dist/`                             |
| `npm run preview`| Preview the build locally before deploying                     |

## Project Structure

- `src/content/post/` — Blog posts (Markdown/MDX)
- `src/site.config.ts` — Site metadata and settings
- `src/styles/global.css` — Theme colors and fonts
- `src/components/` — Astro components
- `public/` — Static assets (favicon, images, manifest)
